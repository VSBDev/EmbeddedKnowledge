# Model-choice practice

## Try the set

:::{callout}
:kind: practice
:id: practice-pair-check

For each reaction, first identify the tracked change. Then label the relevant donor and acceptor and justify the model you chose.

1. `HCl + H2O -> H3O+ + Cl-`
2. `BF3 + NH3 -> F3B-NH3`
3. `NH4+ + H2O <=> NH3 + H3O+`
:::

## Feedback path

- If your pair members do not differ by one proton, redraw the before-and-after formulas side by side.
- If no proton moves, ask which species accepts an electron pair rather than forcing Brønsted–Lowry labels.
- If you chose a model but did not justify it, name the transferred particle or electron pair explicitly.

## Common error

:::{misconception}
:id: misconception-strong-charge

A negative charge does not automatically make a species a strong base, and a positive charge does not automatically make a species a strong acid. Structure, solvent, and equilibrium matter. Charge is useful bookkeeping, not a complete strength rule.
:::

## Evidence boundary

:::{source-note}
:claims: claim-bronsted-model, claim-lewis-model
:sources: source-openstax-chemistry, source-iupac-gold-book

The practice set reuses the two sourced model definitions. It does not treat charge alone as a claim about acid or base strength.
:::
