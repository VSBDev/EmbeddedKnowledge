# The endpoint is not the change

A fictional process is recorded at two times:

| Time, $t$ (min) | Output, $Q$ (units) |
| ---: | ---: |
| 0 | 12 |
| 4 | 28 |

A report calls the average rate $28/4=7$ units per minute. But 28 is the endpoint, not the change. The output changed by $28-12=16$ units, so

$$\frac{\Delta Q}{\Delta t}=\frac{28-12}{4-0}=4\text{ units/min}.$$

A graph is not decoration: each point asserts an input-output pair, and its axes determine what a slope or intercept means.

## Starting check

- In $f(3)=11$, which number is the input?
- If a graph contains $(2,-5)$, what does that say about the function?
- Does $f(x-4)$ move the graph of $f(x)$ left or right?

If coordinate order is uncertain, always read $(x,y)$ as “input first, output second.”

## Accessibility and alternatives

Every graph relationship is supplied as coordinates, equations, interval language, or a value table. No task requires color, visual slope estimation, pointing, dragging, or timing.
