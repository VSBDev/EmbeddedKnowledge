# Fade the supports

Attempt each stage before reading its feedback. The support decreases; the final case leaves the decisions to you.

## Stage 1: completion problem

**Need:** Find a synthesis of studies about whether short pre-laboratory quizzes improve practical performance among first-year biology learners.

**Given:** Answer product = systematic review. Blocks = `[first-year biology learners] [pre-laboratory quizzes] [____]`.

Supply the missing block and name a plausible discovery environment.

**Feedback.** The missing idea is practical laboratory performance, expressed with a candidate such as `practical performance` or `laboratory skill performance`. PubMed could be a starting discovery environment for candidate biomedical-education records. If you added “systematic review” as a topical block, separate document type from subject concepts and use the document type as a later filter or eligibility criterion.

## Stage 2: supported problem

**Need:** Locate original methods and results from studies that measured how microscopy-image practice affects cell-identification accuracy in undergraduate biology.

Use these prompts:

1. What answer product reports original methods and results?
2. Which source or discovery environment can locate candidates?
3. What three concepts determine relevance?
4. If the sample is dominated by medical-residency studies, which block should change first?

**Feedback.** A primary research report matches the answer product. A bibliographic database can locate candidates. Defensible blocks represent undergraduate biology learners, microscopy-image practice, and cell-identification accuracy. If residency studies dominate, make the population block more specific while leaving the other blocks stable. The next action is to inspect whether the revised expression improves relevance, not to assume the retrieved studies are sound.

## Stage 3: independent problem

A course designer needs a current orientation to cognitive load before deciding which narrower research question to ask. Construct the answer-product statement, select a source type, draft two to four concept blocks, and specify one signal that would make you revise a block.

**Feedback criteria.** A current textbook, handbook, or other reference work is a defensible orientation source. The concepts must represent cognitive load and the relevant learning context without smuggling “best,” “effective,” or the intended use into topical blocks. A revision signal must point to a missing or overbroad concept, not merely a disappointing count.

## Stage 4: mixed selection

For each need, choose the best starting product or environment and justify the function.

| Need | Choose from |

**Feedback.** The functional matches are reference work, systematic review, primary report, guideline, and bibliographic database. If you selected the database for every row, you chose a container or discovery route where the need asked for a document product.

## Retry and remediation

On a retry, use a new context rather than copying these answers. Diagnose the error first:

- wrong product: restate “I need ___ in order to ___”;
- wrong source function: compare the source-purpose table;
- missing concept: underline what changes eligibility;
- meaning-changing variant: split it into a new block or remove it;
- too many constraints: remove one nonessential block;
- quality claim from ranking: return to the misconception repair.
