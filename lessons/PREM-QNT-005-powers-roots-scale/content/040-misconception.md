# A root can hide a sign decision

:::{misconception}
:id: misconception-root-square

It is tempting to cancel the square and root in $\sqrt{x^2}$ and write $x$. But the principal square root is nonnegative:

$$\sqrt{x^2}=|x|.$$

For $x=-4$, the left side is $\sqrt{16}=4$, not $-4$.

Likewise, solving $x^2=16$ requires $x=4$ **or** $x=-4$. Applying the principal-root symbol reports 4; solving the equation recovers both signed candidates unless the domain removes one.
:::

This distinction separates an operation from a solution set. Always substitute candidates into the original equation and then apply contextual restrictions.
