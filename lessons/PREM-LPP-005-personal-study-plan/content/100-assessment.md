# Study-plan mastery evidence

Complete the lesson assessment items without opening their answer records until after each attempt. The assessment is untimed and retakeable.

The four tasks require you to:

1. repair an overfilled plan without inventing capacity;
2. build a complete week from constraints, dependencies, a fixed supervised lab, error evidence, a due retrieval contact, and an access boundary;
3. choose evidence-based next-week revisions rather than carrying every task forward; and
4. transfer the planning model to a week with intermittent connectivity.

Use a chronological text ledger, table, calendar, or equivalent structured response. Every route must expose the same decisions: capacity, fixed work, dependencies, learning functions, evidence, contingency, and revision. Do not provide personal health, disability, work, or caregiving details; use the fictional evidence supplied.

## Mastery rule

The response record must show all of these relations across the open tasks:

- the stated capacity boundary is respected;
- concept learning, problems, supervised lab work or preparation/debrief, delayed review, and recovery are all represented;
- prerequisites and preparation precede dependent work;
- at least one later retrieval contact asks for an attempt before feedback;
- recovery capacity or a defensible reduced route is explicit;
- the named error changes a later action; and
- incomplete work is protected, moved, split, replaced, or removed for an evidence-linked reason.

No single category can compensate for a missing dependency or an invented capacity block.

## Feedback and remediation

After each attempt, compare the response with the item reasoning and analytic rubric:

- If capacity is exceeded, rebuild only the boundary and remove or defer the lowest-priority flexible work.
- If categories are present but order is wrong, annotate every prerequisite and fixed event, then resequence.
- If review is only rereading or immediate exposure, add a later reconstruction before feedback and name its target.
- If recovery is only a label on scheduled work, create genuinely uncommitted capacity or state the reduced route.
- If all misses move forward, apply protect, move, split, replace, or remove to each one.
- If access changes the assessed operation, build a text/offline equivalent that preserves the reasoning.

Retry with a new fictional constraint and error record. Do not replay the same answer sequence. If the evidence-to-action chain remains unclear, return to the prerequisite outcome, Metacognition and error logs (PREM-01.04); if retrieval and spacing remain unclear, return to the earlier outcome, Mastery, retrieval, and spacing (PREM-01.03).

Passing this assessment supports the mapped planning outcome within the lesson's conditions. It does not prove a plan was followed, establish delayed mastery, guarantee performance, or authorize clinical or laboratory activity.
