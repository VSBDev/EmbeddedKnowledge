# Demonstrate classification, calibration, and action

Complete the four items in `assessment.json`. On the first attempt, use no notes and commit each response before viewing answer logic. The assessment is untimed and retakeable. An equivalent retake changes the fictional rules, values, surface wording, and error boundary.

Each assessment-rubric criterion is all-or-none: award its full `maxPoints` only when every requirement in that criterion description is satisfied; otherwise award zero for that criterion. This rule makes the item totals reproducible while still separating the four dimensions in each 12-point open response.

## Item map

1. `item-classify-selection` asks you to locate a first broken link from visible intermediate work.
2. `item-calculate-gap` asks you to calculate and interpret a signed confidence–performance gap.
3. `item-build-error-log` asks you to complete the full evidence → diagnosis → action → probe chain across two contrasting records.
4. `item-transfer-review-system` asks you to redesign a score-only weekly review under privacy and access constraints.

## Mastery evidence

A mastery-ready response must:

- distinguish observations from a provisional category;
- locate the earliest unsupported operation or record a genuine unresolved boundary;
- interpret confidence against performance on the same declared scale;
- choose an action that directly exercises the proposed broken link;
- specify new evidence that could confirm or revise the classification; and
- preserve an equivalent untimed text route without personal data.

Correct category words without evidence do not satisfy the analytic rubrics. A calculation without task-bounded interpretation is incomplete. A proposed action without a fresh probe cannot show whether repair occurred.

## Feedback and retry

Use each item's `commonErrors` to identify the first failing distinction: score versus process evidence, confidence versus performance, category versus trait, broad restudy versus targeted action, or activity versus fresh probe. Return to the matching scene, construct a new fictional record that crosses that boundary, and retry. Do not memorize option IDs or lower the reasoning standard.
