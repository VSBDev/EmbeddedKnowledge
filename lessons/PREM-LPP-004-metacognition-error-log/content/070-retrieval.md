# Reconstruct the decision chain

Cover or leave the earlier scenes. Answer each prompt before reading the criterion.

:::{check}
:id: check-reconstruct-chain

1. Write the five operations in order.
2. Define “first broken link” without using a personal label.
3. A learner predicts earning 70% of the declared rubric points and then earns 45%. Calculate and interpret the signed gap.
4. State why a targeted action is incomplete without a fresh probe.
5. If visible work supports either a selection or execution error but does not discriminate them, what should the log record?
:::

## Criterion after the attempt

The chain is interpret → retrieve → select → execute → verify. The first broken link is the earliest operation whose required evidence is absent or contradicted in this record. It is a provisional task diagnosis, not a claim about the learner.

The calibration calculation is $70-45=+25$ percentage points: confidence exceeded rubric performance on this task. The result does not locate the error or establish a stable trait.

A targeted activity is a control decision; a fresh probe produces the next monitoring evidence. Without the probe, the log records an intention but cannot test whether the proposed repair changed performance. When two categories remain possible, record the boundary explicitly and choose a probe that exposes the missing intermediate step—for example, require the learner to state the model choice before executing it.

If fewer than four answers were defensible, revisit `scene-concept` and `scene-calibration`, reconstruct the chain again from memory, and use a new pair of confidence and performance values. If all five were defensible, continue without repeating the page.
