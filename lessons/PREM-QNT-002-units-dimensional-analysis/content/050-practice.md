# Choose, orient, cancel, check

Attempt each item before reading its feedback. On paper or in plain text, show four lines: **target**, **factors**, **surviving unit**, **magnitude check**.

## 1. One factor

Convert $4.6\text{ kg}$ to grams using $1\text{ kg}=1000\text{ g}$.

**Feedback:** $4.6\text{ kg}(1000\text{ g}/1\text{ kg})=4600\text{ g}$. If kilograms remain, reverse the factor.

## 2. A denominator conversion

Convert $3.0\text{ L/min}$ to litres per second.

**Feedback:** $3.0\text{ L/min}(1\text{ min}/60\text{ s})=0.050\text{ L/s}$. A per-second number should be smaller because each second is a smaller time interval than a minute.

## 3. Powered units

Convert $2.5\text{ cm}^3$ to cubic millimetres using $1\text{ cm}=10\text{ mm}$.

**Feedback:** $2.5\text{ cm}^3(10\text{ mm}/1\text{ cm})^3=2500\text{ mm}^3$. Using only 10 leaves uncancelled square centimetres.

## 4. Dimensional audit

A proposed model is $x=at$, where $x$ is distance, $a$ has unit `m/s²`, and $t$ has unit `s`. Can it be the complete equation for distance?

**Feedback:** No. The right side has `m/s`, the dimension of speed. Dimensional inconsistency proves the proposed equation cannot be complete. Adding another factor of time would repair the dimension but would not by itself prove a particular physical model.

## Recovery route

- Target unit cancels: invert the relevant factor.
- Mixed units remain: the chain is incomplete.
- Correct unit but implausible scale: inspect prefixes and denominators.
- Squared or cubed unit remains partly uncancelled: apply the factor to the full power.
- Dimensions agree but context still seems wrong: remember that the audit is necessary, not sufficient.
