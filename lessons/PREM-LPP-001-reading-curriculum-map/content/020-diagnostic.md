# Read before learning the legend

The following miniature map is invented for this diagnostic. Its labels do not describe the Premed course.

```text
Domain: Sciences
  Module: Cells
    Topic A: membrane model
    Topic B: diffusion

Topic A --precedes?--> Topic B
Topic B --related?--- Topic C: osmosis
```

Without checking later scenes, write brief answers:

1. Which label identifies the smallest assessable learning target?
2. What extra information would you need before treating either line as a required study order?
3. If Topic A is secure, what would justify choosing Topic B next?
4. Which part of the map would you use to check whether this cluster belongs to your planned route?

Record confidence as low, medium, or high. Confidence is not a score; it will help you notice whether a later correction changes both your answer and your certainty.

## Recovery route

If terms such as *module*, *prerequisite*, or *route* are unfamiliar, continue to **The map's grammar**. The diagnostic assumes no prior map-reading skill. If the nested text is difficult to parse, rewrite it as three lines—domain, module, topic—and answer from that list.
