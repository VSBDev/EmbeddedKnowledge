# Transfer the audit to a field sensor

A fictional field team is evaluating two filter cartridges for a nitrate sensor used in a stream-monitoring project. No water is collected and no sensor is operated for this lesson.

The team fits either cartridge A or cartridge B, exposes the sensor to the same supplied nitrate standard, and records the sensor reading after five minutes. It keeps standard volume, stated nitrate concentration, sensor model, exposure time, and housing temperature alike. Every cartridge A run uses an older battery pack; every cartridge B run uses a new battery pack.

The team also records:

- a nitrate-free standard that should produce a near-baseline reading; and
- a second certified nitrate standard that should produce a response within a stated range.

Before reading on, make a six-row audit. Identify each role, then answer these questions:

1. What ambiguity does the battery pairing create?
2. How would you break that pairing without changing the cartridge question?
3. If the certified nitrate standard produces a near-zero reading, what can you conclude about an unknown near-zero reading?
4. If the nitrate-free standard produces a large reading, what problem becomes visible?

## Compare your audit

- **Independent variable:** cartridge type, A or B.
- **Dependent variable:** sensor reading after five minutes.
- **Controlled variables:** standard volume, nitrate concentration, sensor model, exposure time, and housing temperature.
- **Confounder:** battery age, because old batteries occur only with cartridge A and new batteries only with cartridge B, while battery condition could change the sensor reading.
- **Negative control:** the nitrate-free standard, expected to show the baseline.
- **Positive control:** the certified nitrate standard expected to produce a response in the stated range.

Use old and new battery packs with both cartridge types, or otherwise arrange battery condition so it is not fixed to cartridge type. This makes cartridge and battery explanations separable while preserving the intended comparison.

If the positive control is near zero, the system has not demonstrated that it can reveal a nitrate response, so an unknown near-zero reading is not interpretable as “no nitrate.” If the negative control gives a large reading, background signal, contamination, or another failure is visible; unknown high readings should not be trusted until that problem is resolved.

This transfer does not ask you to choose a field-study design or make a causal claim about real streams. The next outcome distinguishes experimental from observational designs. Here, the task is narrower: identify the jobs inside a supplied comparison and explain which interpretations those jobs permit.

## Carry the whole chain forward

Complete this sequence from memory:

> observation → bounded empirical question → hypothesis → conditions → prediction → ___

One useful ending is:

> **role audit: changed, recorded, held alike, confounder check, negative control, positive control**

The first lesson separated observations from explanations and built answerable questions. The second turned hypotheses into predictions. This lesson asks whether the comparison distinguishes the prediction from a failed system or a rival factor.

## Mastery check

The assessment contains four untimed, retakeable items worth twelve points. Attempt every item before opening its answer and reasoning. Mastery means earning all twelve points: three for independent, dependent, and controlled roles; three for distinguishing a controlled variable, control condition, and confounder; two for interpreting positive and negative controls; and four for an integrated transfer audit. If you miss a criterion, follow its feedback and retry with the alternate context rather than copying the original answer.

## Accessibility and alternatives

Every task can be completed in prose, bullets, or a table. You may type, dictate into your own device, or use assistive technology; no audio or image upload is required. All table relationships are also explained in nearby prose. The lesson does not depend on color, visual position, dragging, sound, speed, wet-laboratory work, field sampling, or access to equipment. The cases contain no real participant, patient, or environmental data.

## Recovery route

If independent and dependent variables blur together, write **set differently → record response** and place one case detail over each phrase.

If a controlled variable looks like a control group, ask whether the feature is held at one value across the main groups or supplied as an extra comparison. Held alike means controlled variable; extra comparison means control condition.

If every nuisance factor looks like a confounder, apply both tests: it must track the independent variable, and it must be able to affect the dependent variable.

If positive and negative controls reverse, ignore the names at first. Ask **should respond?** for the positive control and **should stay at baseline?** for the negative control.

If a control fails, stop interpreting the unknown. State which capability or background check failed, repair that part, and use a new attempt.
