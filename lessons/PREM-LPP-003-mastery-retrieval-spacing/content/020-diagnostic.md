# Activate your diagnostic baseline

Use one target from your prior `topic-orientation-diagnostic-baseline` work. Do not reopen the source yet.

1. Name the target knowledge in one line.
2. State the future performance you care about: for example, explain a relation, select a model, calculate a result, interpret data, or construct a representation.
3. Mark your most recent unaided evidence as **unfamiliar**, **fragile**, or **secure**.
4. Write the observation behind that label. Record what you produced, what prompts or notes were available, when the attempt occurred, and which part failed or succeeded.

Do not upgrade a classification because the page looks recognizable now. Preserve the earlier observation so the new plan has a genuine starting state.

## Route decision

Continue if you can state both a target performance and a recent unaided observation. Keep your classification available for the worked example.

## Recovery route

If you cannot state a target performance or have no unaided observation, return to `topic-orientation-diagnostic-baseline` (`PREM-01.02`). Complete one low-stakes closed-source attempt, compare it with the relevant criteria, and record the result. Then restart this scene. The route supplies missing planning information; it is not a penalty or a grade.

If the word **secure** appears in your baseline but the record contains only rereading, copying, or following a displayed solution, leave the label undecided and collect an unaided attempt before continuing.
