# Show mastery and schedule the next retrieval

The mastery check has four retakeable items worth 12 points. Attempt them without returning to the worked example.

You have demonstrated this lesson's outcome when you earn at least 9 of 12 points **and** at least 3 of 4 points on the chamber-transfer item. That conjunctive rule prevents a strong calculation from hiding a weak domain or failure analysis.

The items ask you to:

1. judge what a successful fit can support;
2. use range and boundary evidence to diagnose a linear model failure;
3. calculate an ideal-gas prediction and a local sensitivity range; and
4. audit a scale transfer and justify a restrict, revise, or replace decision.

## Remediation by error pattern

- If you cannot name inputs, parameters, assumptions, or output, return to **Four questions before trusting a model** and rebuild the audit table for one model from another course.
- If the formula use is correct but the interpretation is overconfident, return to **Predict pressure, then test the prediction** and add one sentence beginning “This result is conditional on…”
- If interpolation, extrapolation, or sensitivity is unclear, redo the feeder-count and instrument cases in **Make the model show where it breaks** with different boundary values.
- If you notice a mismatch but cannot choose a response, return to the chamber transfer and ask whether the problem lies in the parameter values, the represented structure, or the question being asked.

Every retry uses a changed context described in the answer reasoning. Do not memorize an option sequence. Reconstruct the audit and explain why the decision would change if the purpose or domain changed.

## Delayed retrieval

In about 48 hours, choose a model from biology, chemistry, physics, or statistics and write five lines from memory: purpose/output, inputs/parameters, assumption, scale/domain, and failure test.

One week later, revisit this prompt before studying mechanism evidence:

> Two models match the same observed output. What new prediction would discriminate between them, and what would each possible result justify?

That question reconnects the earlier hypothesis–prediction prerequisite to the next scientific-inquiry task: deciding when evidence supports a proposed mechanism rather than merely an output fit.

## Final synthesis

A model is productive when it earns a bounded conditional claim:

> Under these assumptions, at this scale, and inside this domain, the model gives this output with these known failure tests.

When a test fails, name the location of failure. Restrict the claim if the use outran the evidence. Recalibrate if defensible parameter values are the problem. Revise the structure if a necessary relationship is missing. Replace the model when another representation better serves the question.
