# Reconstruct the reference-first method

1. Attach labels and preserve ratio order.
2. State why proportional scaling is justified.
3. Name the whole or starting value that supplies the denominator.
4. Calculate with a fraction before converting to percent.
5. Reverse, substitute, or bound the answer.

## Retrieval

- Why are $3:5$ and $6:10$ equivalent?
- What assumption makes a proportion appropriate?
- In percent change, why is the old value the denominator?
- How do percentage points differ from relative percent change?

Then complete `assessment.json`. If ratio order fails, return to “Keep the ratio labels attached.” If the wrong denominator appears, redo the mixture transfer with every whole written explicitly. If arithmetic is correct but the comparison is unlabeled, add “percentage points” or “percent increase/decrease” before retrying.
