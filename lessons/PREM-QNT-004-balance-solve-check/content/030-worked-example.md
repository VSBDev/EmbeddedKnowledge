# Solve without changing the constraint

A fictional counter follows $5(2x-3)+4=3x+31$. Find $x$ and verify it.

:::{worked-example}
:id: worked-example-linear-both-sides

**1. Declare the domain.** No denominator or contextual restriction is stated, so use real numbers.

**2. Expose structure.** Distribute 5 across both terms:

$$10x-15+4=3x+31,$$

so $10x-11=3x+31$.

**3. Preserve equality.** Subtract $3x$ from both sides and add 11 to both sides:

$$7x=42.$$

Divide both sides by nonzero 7: $x=6$.

**4. Substitute into the original.** Left side: $5(12-3)+4=49$. Right side: $18+31=49$. Both sides agree.

**5. Challenge the scale.** Before exact work, coefficients suggest a small positive value; the checked result is consistent.
:::

:::{check}
:id: check-original-equation

Why is checking the original equation stronger than checking only the final line $7x=42$?
:::

An earlier distribution or sign error could create a valid solution to the wrong transformed equation. The original retains every initial term and grouping.
