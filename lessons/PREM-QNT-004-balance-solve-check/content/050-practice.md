# Fade from operations to justification

Attempt each before reading feedback. Record the operation applied to both sides.

## 1. Two-step equation

Solve $7x+5=47$.

**Feedback:** Subtract 5, then divide by 7: $x=6$. Check: $42+5=47$.

## 2. Parentheses

Solve $4(x+3)=2x+22$.

**Feedback:** $4x+12=2x+22$, so $2x=10$ and $x=5$. Both original sides equal 32.

## 3. Fractions

Solve $x/5-(x-2)/3=2$.

**Feedback:** Multiply every term by 15: $3x-5(x-2)=30$. Then $-2x+10=30$, so $x=-10$. Original check: $-2-(-12/3)=2$.

## 4. Classify

Solve $2(x+4)=2x+8$.

**Feedback:** Both sides reduce to $2x+8$; the result $0=0$ means every real $x$ satisfies the identity.

## Recovery route

- Sign error: write the same addition or subtraction explicitly on both sides.
- Lost term after clearing fractions: multiply **every** term by the common denominator.
- Contradiction mistaken for a value: stop when the variable cancels and classify the remaining statement.
- Candidate fails substitution: return to the first line where transformed sides ceased to be equivalent.
