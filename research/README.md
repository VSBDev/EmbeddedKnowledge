# Research

Working notes behind the Premed curriculum. Published so the curriculum's sources and
reasoning can be inspected, not as finished scholarship.

- `SOURCE-CATALOG.md` — the cited source catalog. Findings that survived adversarial
  verification are marked; later corrections are annotated inline.
- `DIARY.md` — the process log, including the method lessons that shaped how curriculum
  research is run.
- `PORTABLE-RESEARCH-PROMPT.md` — the reusable research brief and adjudication prompt, so
  the same method can be repeated for another subject.
- `CONTENT-AUTHORING-EVIDENCE-2026-07-19.md` — evidence base for the authoring standard.

## What is deliberately not published

Raw model runs and the adjudications between them are kept locally as working evidence.
They contain source citations that were never verified — some were wrong, and at least one
set of sources was never checked at all — so publishing them would present unchecked claims
as project research. Where those runs produced a conclusion the project relies on, that
conclusion appears in `SOURCE-CATALOG.md` with its verification status stated.
