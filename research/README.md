# Research

Working evidence behind EmbeddedKnowledge curricula. Published so source selection,
rights decisions, and design reasoning can be inspected, not as finished scholarship.

- `SOURCE-CATALOG.md` — the cited source catalog. Findings that survived adversarial
  verification are marked; later corrections are annotated inline.
- `DIARY.md` — the process log, including the method lessons that shaped how curriculum
  research is run.
- `PORTABLE-RESEARCH-PROMPT.md` — the reusable research brief and adjudication prompt, so
  the same method can be repeated for another subject.
- `CONTENT-AUTHORING-EVIDENCE-2026-07-19.md` — evidence base for the authoring standard.
- `PSYCHIATRY-SOURCE-CATALOG.md` — the dated training-framework, program-practice,
  open-knowledge, rights, and design synthesis behind the Psychiatry research candidate.

## What is deliberately not published

Raw model runs and intermediate adjudications are kept locally as working evidence. They
may contain claims or citations that have not survived verification, so publishing them
would present unchecked material as project research. Where a run produced a conclusion
the project relies on, that conclusion appears in the relevant source catalog with its
method, primary source, date, rights posture, and limitations stated.
